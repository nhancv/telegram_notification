
## How to use the script

```
TELEGRAM_CHAT_ID=
TELEGRAM_MESSAGE=
TELEGRAM_BOT_TOKEN=
curl -L -s https://raw.githubusercontent.com/nhancv/telegram_notification/master/telegram_notification.sh | bash /dev/stdin "$TELEGRAM_CHAT_ID" "$TELEGRAM_MESSAGE" "$TELEGRAM_BOT_TOKEN"
```
